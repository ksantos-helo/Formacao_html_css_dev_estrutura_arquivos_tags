```html
<main>

```

## 1️⃣ Título
```
html
Copiar código
<h1>
  Eleve seu negócio digital a outro nível
  <strong>com um Front-end de qualidade!</strong>
</h1>
<strong> destaca a parte importante do texto.
```

## 2️⃣ Parágrafo
```
html
Copiar código
<p>Olá! Sou Joana Santos, desenvolvedora Front-end com especialidade em React, HTML e CSS.
Ajudo pequenos negócios e designers a colocarem em prática boas ideias. Vamos conversar?</p>
```

## 3️⃣ Links (botões)
```
html
Copiar código

```
html
