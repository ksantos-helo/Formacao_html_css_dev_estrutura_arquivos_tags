# Estrutura Básica de um Arquivo HTML

- Todo arquivo HTML possui uma **estrutura padrão** que deve ser seguida.  
- É importante conhecer as **tags semânticas**, que descrevem o conteúdo que armazenam e ajudam na organização do código.  
  - Exemplos: `<header>`, `<main>`, `<footer>`.  
- Vantagens das tags semânticas:
  - Facilitam a **leitura do código** por outros desenvolvedores.  
  - Melhoram a **interpretação pelos navegadores** e mecanismos de busca.  

> Para aprofundar, consulte a documentação MDN sobre [Semântica em HTML](https://developer.mozilla.org/pt-BR/docs/Glossary/Semantics) e explore outros elementos semânticos que tornam seu código mais claro e acessível.
