# Aprendizado: HTML e CSS

## 1. Estrutura e Estilização
- HTML cria a **estrutura semântica** da página (títulos, parágrafos, listas, imagens, links).
- CSS é responsável pela **estilização visual**: cores, tamanhos, fontes, posicionamento e layout.
- HTML e CSS **não são linguagens de programação**, mas sim de marcação e estilo.

## 2. Diferença entre HTML e CSS
- HTML: descreve **o que** aparece na página.
- CSS: descreve **como** os elementos devem aparecer.
- Separar estrutura (HTML) de estilo (CSS) facilita manutenção e padronização do design.

## 3. Histórico e importância do CSS
- Antes, HTML continha marcação e estilização juntas, gerando arquivos confusos.
- CSS foi criado pelo **W3C** para separar o estilo da estrutura.
- Arquivos CSS externos permitem alterar a aparência de várias páginas modificando apenas um arquivo.

## 4. Aprendizado prático
- Consultar documentação do CSS (ex.: W3Schools) é importante para aprender gradualmente.
- É possível aplicar **múltiplos estilos** a um mesmo HTML usando diferentes arquivos .css.
- Focar nas propriedades principais primeiro, sem tentar memorizar todas de uma vez.
